# Ejecutar en desarrollo
1. clonar el repo 
2. Ejecutar el siguiente comando... 
````
yarn install
````
3. tener Nest CLI instalado... 
````
npm install i -g @nestjs/cli
````
4. levantar la base de datos... 
````
docker-compose up -d
````

## Stack usado
* MongoDB
* Nest
